## Resultado

Ruta nueva `GET /expedientes/{expediente_id}/presupuesto-reparacion`:
- muestra datos básicos de expediente;
- lista patologías con coste asociado;
- muestra estancia, patología, partidas, cantidad, unidad, precio unitario e importe;
- calcula subtotal por patología;
- calcula total PEM de reparación;
- muestra resumen por capítulo/familia si hay capítulo o por prefijo de código si no;
- muestra estado vacío si no hay costes vinculados.

Se añadió enlace local desde detalle de expediente para informes de patologías. No se modifica el informe ni se exporta nada.
