## Resultado

Se anade `POST /costes/descompuestos/{descompuesto_id}/actualizar` para editar
lineas existentes. Los cambios de descompuestos devuelven la partida a
`borrador`. Si una partida validada cambia codigo o precio, tambien vuelve a
`borrador` con aviso.

Las pantallas de revision de captura y detalle de partida usan tablas desktop
reales para descompuestos, con resumen ancho, columnas numericas visibles y
acciones por fila.
