# Failures

Memoria de fallos, warnings o acoplamientos que ya aparecieron. El objetivo es no redescubrir el mismo problema en cada sesion.

## Formato

- Que paso.
- Causa raiz.
- Deteccion.
- Impacto.
- Mitigacion.
- Como evitar regresion.
- Smoke tests relacionados.

## Regla

Registrar aqui fallos relevantes que cambien como Codex debe trabajar, validar o planificar.
